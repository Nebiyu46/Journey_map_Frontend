# Journey Map

A React application for visualizing student journey maps.

## Features
- Interactive, hierarchical step visualization
- Expandable/collapsible nodes
- Status tracking (To Do, In Progress, Completed)
- Dark/Light mode support

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
